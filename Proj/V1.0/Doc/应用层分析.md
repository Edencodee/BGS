# ISL700_2A背景抑制光电传感器应用技术报告
## 1. 产品概述

ISL700_2A是一款高度集成的CMOS光电传感器芯片，专用于基于双PD三角测距原理的背景抑制（BGS）光电传感器。芯片集成两个光电二极管（PD）及相应的信号处理电路，支持可编程参数调节，适用于前景检测、距离抑制等多种应用场景。

### 1.1 主要特性
- 双PD结构：A管（600μm × 800μm）、B管（600μm × 1960μm）
- 可编程参数：增益、发射光强、时钟频率、同步控制
- 支持多传感器同步，避免干扰
- 输出类型：NPN开集电极，支持亮通/暗通逻辑切换
- 工作电压：3.3V～6.5V
- 工作温度：-40℃～85℃

---

## 2. 三角测距原理与信号定义

### 2.1 双PD工作原理
两个PD位于不同成像位置，接收同一反射光。目标在不同距离时，两个PD接收到的光能分配不同，通过计算其比例或差分可判断距离。

### 2.2 距离参数定义
推荐使用以下公式计算距离参数：

R = V_B - V_A  /  V_B + V_A + epsilon


其中：
- V_A：A管信号电压（对应 `sh_out` 在 `pls_so=0`）
- V_B：B管信号电压（对应 `sh_out` 在 `pls_so=1`）
- epsilon：防止除零的小常数（如 0.001）

### 2.3 信号采集时序
- `sh_out` 输出为A/B交替的模拟电压
- `pls_so=0`：输出V_A
- `pls_so=1`：输出V_B
- 每个通道信号为2个PLS周期的平均值

## 3. 示教（Teach-in）功能实现流程

### 3.1 机械调焦（旋钮调节）
旋转透镜改变光学成像关系，粗略设定检测距离。

### 3.2 软件示教（微调）
1. 将目标物置于期望触发位置。
2. 采集多组V_A、V_B，计算 R 的平均值，记为R_set。
3. 设定触发与复位阈值（带迟滞）：
   TH_ON = R_set + margin
   TH_OFF = R_set - margin

4. 将R_set、TH_ON、TH_OFF 存入非易失存储器。

> **margin 建议值**：可取R_set 的 2%~5%，或基于噪声标准差 3 sigma 计算。

---

## 4. 输出逻辑与NPN控制

### 4.1 NPN输出特性
- 导通：输出下拉至 GND（低电平，吸电流）
- 断开：输出悬空（外部上拉至高电平）

### 4.2 亮通（Light-ON）与暗通（Dark-ON）
| 模式 | 检测条件满足时输出 | 逻辑表达式 |
|------|------------------|------------|
| 亮通 | 导通（低电平） | `OUT = (R > TH_ON)` |
| 暗通 | 导通（低电平） | `OUT = !(R > TH_ON)` |

> 实际控制中可通过一个 `invert` 标志实现：
> `OUT_active = (R > TH_ON) XOR DARK_MODE`

## 5. 软件处理流程图

```plaintext
初始化
├─ 读取保存的 R_set, TH_ON, TH_OFF
├─ 配置寄存器（增益、时钟、发射强度）
└─ 进入主循环

主循环
├─ 采集 V_A, V_B（通过 sh_out + pls_so）
├─ 计算 R = (V_B - V_A) / (V_B + V_A + ε)
├─ 判断目标状态：
│   ├─ 若 R > TH_ON → 目标在检测区内
│   ├─ 若 R < TH_OFF → 目标离开
│   └─ 否则保持原状态
├─ 输出控制：
│   ├─ Light-ON: 目标在区内 → NPN导通
│   └─ Dark-ON: 目标不在区内 → NPN导通
└─ 更新状态LED指示
