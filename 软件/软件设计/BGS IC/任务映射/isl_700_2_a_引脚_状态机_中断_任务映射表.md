# 🧩 ISL700_2A 引脚 → 状态机 → 中断 → 任务映射表

## 1. 文档目的

本文档用于建立 ISL700_2A **硬件引脚、软件状态机、中断机制、后台任务** 之间的一一映射关系，用于：
- 防止驱动与应用层职责混乱
- 统一中断与任务的边界
- 指导系统调试、异常定位与评审

本表是 **软件架构与硬件接口的核心桥梁文档**。

---

## 2. 全局设计原则

1. **任何硬件引脚都必须能映射到明确的软件责任单元**
2. **中断只做“采集与标记”，不做算法与判定**
3. **状态机只切换状态，不直接操作硬件细节**
4. **任务层完成最终业务闭环（算法 → 输出）**

---

## 3. 系统状态机定义

| 状态 | 描述 |
|---|---|
| INIT | 上电初始化 |
| CONFIG | IC 参数配置 |
| IDLE | 空闲/待机 |
| RUN | 正常测量 |
| ERROR | 错误处理 |

---

## 4. 引脚 → 状态机 → 中断 → 任务 映射总表

### 4.1 配置 / 校验域

| 引脚 | 功能 | 状态机归属 | 中断 | 软件任务 |
|---|---|---|---|---|
| Cal_in | 配置数据 | CONFIG | 无 | Config_Task |
| Cal_clk | 配置时钟 | CONFIG | 无 | Config_Task |
| CRC_rst | CRC 复位 | CONFIG | 无 | Config_Task |
| STS | 配置结果 | CONFIG | 可选 EXTI | Config_Check_Task |

**工程说明**
- 仅 CONFIG 状态允许操作上述引脚
- RUN / IDLE 状态禁止重新配置

---

### 4.2 发射 / 时钟控制域

| 引脚 | 功能 | 状态机归属 | 中断 | 软件任务 |
|---|---|---|---|---|
| clk_mode | 时钟模式 | CONFIG | 无 | Config_Task |
| ext_slvclk | 外部时钟 | RUN | Timer | Timing_Task |
| ext_clk_hf | 高频时钟 | RUN | Timer | Timing_Task |
| freq_c | 发射频率 | CONFIG | 无 | Config_Task |
| close_pls | 发射开关 | INIT / ERROR | 无 | Safety_Task |
| pls_delay | 发射输出 | RUN | 可选 EXTI | Timing_Monitor_Task |

**工程说明**
- close_pls 在 INIT / ERROR 状态强制为 1
- pls_delay 仅作为监测与调试依据

---

### 4.3 接收 / 采样域

| 引脚 | 功能 | 状态机归属 | 中断 | 软件任务 |
|---|---|---|---|---|
| pls_so | A/B 指示 | RUN | EXTI | Sample_Sync_Task |
| SH_OUT | 模拟输出 | RUN | ADC/DMA | Sample_Acquire_Task |
| SW_OUT | 调试输出 | DEBUG | 无 | Debug_Task |

**工程说明**
- pls_so 中断是采样系统的时间基准
- ADC 采样必须与 pls_so 逻辑绑定

---

### 4.4 增益 / 模拟前端域

| 引脚 | 功能 | 状态机归属 | 中断 | 软件任务 |
|---|---|---|---|---|
| sctrl_a | 增益控制 | CONFIG | 无 | Gain_Config_Task |
| sctrl_b | 增益控制 | CONFIG | 无 | Gain_Config_Task |

**工程说明**
- 增益参数冻结于 CONFIG 完成后
- RUN 状态禁止动态修改

---

## 5. 中断 → 任务 调用关系

```text
EXTI(pls_so)
   ↓
Sample_Sync_Task
   ↓
ADC_Start
   ↓
ADC/DMA IRQ
   ↓
Sample_Acquire_Task
```

```text
Config_Task
   ↓
STS Check
   ↓
State Transition (CONFIG → IDLE)
```

---

## 6. 状态机 → 引脚控制矩阵

| 状态 | close_pls | Cal_in/clk | ADC | 算法 |
|---|---|---|---|---|
| INIT | 1 | 禁止 | 禁止 | 禁止 |
| CONFIG | 1 | 允许 | 禁止 | 禁止 |
| IDLE | 0/1 | 禁止 | 可选 | 禁止 |
| RUN | 0 | 禁止 | 允许 | 允许 |
| ERROR | 1 | 禁止 | 禁止 | 禁止 |

---

## 7. 典型错误防护规则

- RUN 状态禁止触发 Config_Task
- 未进入 RUN 状态禁止启用 ADC 中断
- 未完成 CONFIG 状态禁止 close_pls=0
